# Tru Skool Mall — 3D Model Shopping List

## Quick Start
1. Visit each URL below (free Sketchfab account required for downloads)
2. Click "Download 3D Model" → select **glTF** format
3. Rename the downloaded .glb file to the filename shown
4. Upload to `assets/models/` in the GitHub repo
5. The mall auto-loads any model it finds — no code changes needed

## After Download: Upload to GitHub

1. Go to https://github.com/truskoolcorp/tru-skool-mall/tree/main/assets/models
2. Click "Add file" → "Upload files"
3. Drag in all your .glb files
4. Commit
5. Vercel auto-deploys — models appear in the mall automatically
